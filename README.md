This is a Version Control Tutorial for those lazy Pisaynons, you're welcome.

Step 1: Get your tools, VS Code, Git, Git Bash/cmd.exe (Command Prompt), and GitHub Acc
Step 2: Create a folder
Step 3: Open VS Code
Step 4: Go to the Top menu, and then press File
Step 5: Press Open Folder, and select the folder you made earlier
Step 6: Initialize repository by clicking the big blue button that says Initialize Repository
Step 7: Make a new file
Step 8: Code something simple
Step 9: Go to Source Control, the branching path icon on the left side
Step 10: Hover over the file name of your file, then click the plus sign / Stage Changes
Step 11: Put a message in the message box
Step 12: Commit changes
Step 13: Open github.com
Step 14: Go to the top right and find a plus sign, click on it and press new repository
Step 15: Give it a name, and do not click any of the weird box stuff, just save it
Step 16: Go to Git Bash or cmd.exe (Which is already installed on Windows) and type the following:
  git config --global user.name *your_username*
  git config --global user.email *your_email*
Step 17: Go back to github.com and scroll to the bottom of the page, then copy paste the three lines of code
Step 18: Go back to the VS Code Top menu and click on Terminal, then New Terminal
Step 19: Paste the lines of code in the terminal
Step 20: If you receive an error, call Ethan Malinao, he'll probably figure it out
Step 21: Click Publish Repository
Step 22: Go back to github.com then keep refreshing the page until it changes
Step 23: Screenshot then send through the submission bin, it's ILA 2
You're done!
